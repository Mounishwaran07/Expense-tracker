# Expense Tracker Web App

A simple **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript**.
This project helps users track their daily expenses easily.

## Features

* Add new expenses
* Edit existing expenses
* Delete expenses
* Search expenses by category
* Download expense report (CSV)
* Dark mode toggle
* Data stored using LocalStorage

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Enter the **Category** and **Amount**
2. Click **Add Expense**
3. View all expenses in the table
4. Use **Edit / Delete** to modify expenses
5. Use **Search** to filter expenses
6. Click **Download Report** to export data

## Data Storage

All expense data is stored in the browser using **LocalStorage**, so it remains saved even after refreshing the page.
